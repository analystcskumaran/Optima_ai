# OPTIMA - AI Data Analyst Assistant

## Overview
OPTIMA is a standalone Python-based AI data analyst assistant for automating CSV data processing, cleaning, analysis, visualization, and EDA.

## Features
- Data ingestion and validation
- Automated cleaning (missing values, duplicates, outliers, types)
- Statistical analysis and correlations
- Visualizations (plots, trends)
- EDA with Sweetviz
- AI-powered query interface using Hugging Face

## Installation
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run app/main.py`

## Usage
Upload a CSV, process it, query via AI, or view reports.
